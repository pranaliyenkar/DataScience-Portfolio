# Hospital Power BI Dashboard

A fully interactive Power BI report providing insights into inpatient,outpatient, day-case, and inpatient waiting lists across specialties, age groups, and time bands.

---

##  Project Overview

This Power BI project visualises NHS waiting list data to help stakeholders understand trends, performance, and pressure points across different case types and specialties.  
The report includes **summary metrics**, **interactive charts**, and a **detailed data table** spanning **January 2018 to November 2019**.

---

##  Key Features

### **1. Summary Page**
- **Latest Month Wait List** and **Previous Year (PY) Latest Month Wait List** KPIs  
- **Donut Chart**: Avg/Median wait list by *Case Type*  
- **Stacked Column Chart**: Wait list distribution by **Time Bands** and **Age Profile**  
- **Trend Lines**:
  - Outpatient monthly totals  
  - Day Case + Inpatient trends  
- **Specialty KPIs**: Avg/Median wait list by specialty  
- **Interactive filters**:
  - Archive Date  
  - Specialty  
  - Case Type  
  - Age Profile  
  - Time Bands  

---

### **2. Detail Page**
- Monthly breakdown of:
  - Day Case  
  - Inpatient  
  - Outpatient  
  - Total count  
- **Date range slider** for filtering  
- Interactive slicers for:
  - Specialty  
  - Case Type  
  - Age Profile  
  - Time Bands  
- Total row summarising all visible records

## Key Questions This Dashboard Answers

- How have waiting lists changed month-over-month?  
- Which specialties have the highest wait times?  
- What age groups are waiting the longest?  
- How do wait lists differ across inpatient, day case, and outpatient services?  
- Are wait times improving compared to last year?
